MCP Todo API Test

Welcome to the MCP Todo Server! Below are the available tools and their usage examples:

createTodo

Creates a new todo item

{
  "name": "createTodo",
  "params": {
    "title": "Review PR #123"
  }
}

listTodos

Lists all todos or filters by completion status

{
  "name": "listTodos",
  "params": {
    "completed": false  // Optional parameter
  }
}

updateTodo

Updates a todo's completion status

{
  "name": "updateTodo",
  "params": {
    "id": "todo-id-here",
    "completed": true
  }
}

deleteTodo

Removes a todo item

{
  "name": "deleteTodo",
  "params": {
    "id": "todo-id-here"
  }
}

Current Database

The todos are stored in db.json. Example structure:

{
  "todos": [
    {
      "id": "unique-id",
      "title": "Review PR #123",
      "completed": false
    },
    {
      "id": "another-id",
      "title": "Update documentation",
      "completed": true
    }
  ]
}

Getting Started

To use these tools:

  1. Enable Agent mode in VS Code
  2. Add this MCP server in VS Code settings
  3. Try the example commands shown in the README

For detailed setup instructions, please refer to the project README.
